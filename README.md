## Getting Started

This is a guide to setup the app localy

### Prerequisites

Firstly you need Node and Yarn
* npm
  ```
  npm install npm@latest -g
  npm install yarn -g
  ```

### Installation

_Installation instructions_

1. Clone the repo
   ```sh
   git clone https://github.com/alkrajnc/imageboard
   ```
2. Install YARN packages
   ```sh
   cd imageboard
   yarn
   yarn run dev
   ```
